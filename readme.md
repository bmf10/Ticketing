# Ticketing

## Installation
- Clone or Download
- If download copy to Xampp/htdocs/ or anything else
- Import DB in database folder
- Run `localhost/ticketing`
- Login Superadmin `localhost/ticketing/admin/auth`
- Email: `superadmin`, Password: `123456`
